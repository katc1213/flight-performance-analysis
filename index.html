Power BI Dashboard & ML Research Reports

How to embed your Power BI report:
In Power BI, go to File → Embed report → Publish to web (public) and copy the <iframe> src URL. Then replace YOUR_POWER_BI_EMBED_URL_HERE in this file with that URL.
How to add your reports:
Upload your files (PDF or DOCX) to the same GitHub repo folder and set each href to the filename (e.g. href="report1.pdf"), or paste a Google Drive / SharePoint share link directly.
